# OS(운영체제) 란 무엇인가요?
## 1. OS의 정의
하드웨어와 사용자(또는 애플리케이션) 사이에서 자원을 관리하고, 프로그램이 실행될 수 있도록 도와주는 시스템 소프트웨어
> "운영체제는 CPU, 메모리, 저장장치 같은 하드웨어 자원을 효율적으로 관리하고, 프로그램이 실행될 수 있도록 환경을 제공하는 소프트웨어입니다."

## 2. OS의 역할
### 1️⃣ 프로세스 관리
프로그램 실행 → 프로세스 생성
CPU 스케줄링
멀티태스킹 지원
예:
카카오톡 + 크롬 + VSCode 동시에 실행 가능

### 2️⃣ 메모리 관리
어떤 프로세스가 얼마만큼 메모리 쓰는지 관리
가상 메모리 제공
메모리 보호

### 3️⃣ 파일 시스템 관리
파일 생성/삭제
디렉토리 구조 관리
권한 관리

### 4️⃣ 입출력(I/O) 관리
키보드
마우스
프린터
네트워크
하드웨어 직접 제어는 OS가 담당

### 5️⃣ 보안 및 권한 관리
사용자 계정
접근 권한
프로세스 격리

## 3. OS의 주요 구성 요소
### 🔹 커널(Kernel)
OS의 핵심
하드웨어 직접 제어
프로세스/메모리 관리

### 🔹 쉘(Shell)
사용자 인터페이스
명령어 해석 - 터미널
GUI / CLI

### 🔹 시스템 콜(System Call)
프로그램 → OS 요청
예: 파일 열기, 메모리 할당

### 🔹 드라이버(Driver)
하드웨어 제어
각 장치별 드라이버 필요

### 🔹 인터페이스
사용자가 OS와 상호작용하는 방법
GUI (Graphical User Interface)
CLI (Command Line Interface)
