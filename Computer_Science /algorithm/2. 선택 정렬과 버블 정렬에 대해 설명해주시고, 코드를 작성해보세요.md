# 버블 정렬과 선택 정렬에 대해 설명해주시고, 코드를 작성해보세요.

## 버블 정렬 (Bubble Sort)
### 개념
인접한 두 요소를 비교해서, 큰 값을 뒤로 보내는 방식의 정렬
한 바퀴를 돌 때마다
👉 가장 큰 값이 맨 뒤로 이동함.

### 동작 과정 예시
```
[5, 3, 2]

1회전:
5 > 3 → 교환 → [3, 5, 2]
5 > 2 → 교환 → [3, 2, 5]

2회전:
3 > 2 → 교환 → [2, 3, 5]
```

### 시간복잡도
평균 / 최악: O(n²)
최선(이미 정렬됨): O(n)

### 특징
구현 매우 쉬움
성능 매우 느림
거의 실무에서는 사용 안 함 (교육용)

```dart
void bubbleSort(List<int> arr) {
  final n = arr.length;

  for (int i = 0; i < n - 1; i++) {
    bool swapped = false;

    for (int j = 0; j < n - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        final temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
        swapped = true;
      }
    }

    if (!swapped) break; // 이미 정렬된 경우 종료
  }
}
```

## 선택 정렬 (Selection Sort)
### 개념
가장 작은 값을 찾아서 맨 앞과 교환하는 방식
매 단계마다
👉 “최솟값을 선택” 해서 정렬 위치로 보냄.

### 동작 과정 예시
```
[5, 3, 2]

1단계:
최솟값 2 → 5와 교환 → [2, 3, 5]

2단계:
최솟값 3 → 그대로
```

### 시간복잡도
평균 / 최악 / 최선 모두: O(n²)
(이미 정렬돼 있어도 다 비교함)

### 특징
교환 횟수는 적음
비교 횟수는 항상 동일
불안정 정렬

### Dart 코드
```dart
void selectionSort(List<int> arr) {
  final n = arr.length;

  for (int i = 0; i < n - 1; i++) {
    int minIndex = i;

    for (int j = i + 1; j < n; j++) {
      if (arr[j] < arr[minIndex]) {
        minIndex = j;
      }
    }

    if (minIndex != i) {
      final temp = arr[i];
      arr[i] = arr[minIndex];
      arr[minIndex] = temp;
    }
  }
}
```

## 차이점 정리 (면접 핵심)
| 구분       | 버블 정렬   | 선택 정렬    |
| -------- | ------- | -------- |
| 방식       | 인접 비교   | 최솟값 선택   |
| 교환 횟수    | 많음      | 적음       |
| 최선 시간복잡도 | O(n) 가능 | 항상 O(n²) |
| 안정성      | 안정      | 불안정      |


## 면접용 정리 답변
버블 정렬은 인접한 요소를 비교하여 큰 값을 뒤로 보내는 방식이고,
선택 정렬은 매 단계마다 최솟값을 선택하여 앞쪽과 교환하는 방식입니다.
두 알고리즘 모두 평균 시간복잡도는 O(n²)이지만,
버블 정렬은 최적화 시 최선의 경우 O(n)이 가능하고,
선택 정렬은 항상 O(n²)의 성능을 가집니다.
