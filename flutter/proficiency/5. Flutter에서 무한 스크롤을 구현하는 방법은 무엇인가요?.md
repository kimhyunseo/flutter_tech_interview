# Flutter에서 무한 스크롤을 구현하는 방법은 무엇인가요?
## ScrollController를 이용한 수동 구현 (가장 일반적인 방법)
ListView.builder와 ScrollController를 조합하여 스크롤 위치를 감시하는 방식

구현 로직

ScrollController 생성: 스크롤 위치를 추적합니다.

리스너 등록: 스크롤이 끝(또는 끝 근처)에 도달했는지 확인합니다.

데이터 요청: 끝에 도달하면 서버 API를 호출하여 다음 '페이지' 데이터를 가져옵니다.

상태 업데이트: 가져온 데이터를 기존 리스트에 추가하고 화면을 다시 그립니다.

---

## 전문 패키지 사용 (infinite_scroll_pagination)
복잡한 프로젝트나 에러 처리, 새로고침 기능 등이 많이 필요하다면 검증된 패키지를 쓰는 것이 정신 건강에 이롭습니다.

장점: '데이터 없음', '에러 발생', '첫 페이지 로딩' 등 다양한 상태 처리가 위젯 하나로 끝납니다.

사용법: PagingController를 선언하고, 데이터 로드 함수를 연결하기만 하면 됩니다.
