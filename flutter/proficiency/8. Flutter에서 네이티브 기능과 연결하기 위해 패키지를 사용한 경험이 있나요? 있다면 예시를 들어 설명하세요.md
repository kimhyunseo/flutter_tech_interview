# Flutter에서 네이티브 기능과 연결하기 위해 패키지를 사용한 경험이 있나요? 있다면 예시를 들어 설명하세요.
## 1. 위치 정보 활용 (geolocator)
사용자의 현재 위치를 파악하여 가장 가까운 야구장을 추천해 주는 기능을 구현할 때 필수적입니다.

활용 사례: "내 주변 경기장 찾기" 메뉴에서 사용자의 위도/경도를 가져와 서버 API로 전달합니다.

패키지의 역할: 안드로이드의 LocationManager와 iOS의 CoreLocation 프레임워크를 개발자가 직접 다룰 필요 없이, Dart 코드 한 줄(Geolocator.getCurrentPosition())로 위치를 가져올 수 있게 해줍니다.

## 2. 카메라 및 갤러리 접근 (image_picker)
사용자가 프로필 사진을 등록하거나, 직관 현장에서 찍은 사진을 게시글에 업로드할 때 사용합니다.

활용 사례: "프로필 수정" 화면에서 카메라로 사진을 찍거나 갤러리에서 선택합니다.

패키지의 역할: 각 OS의 복잡한 권한 요청(Permission) 처리와 이미지 선택 인터페이스를 자동으로 띄워줍니다.

## 3. 외부 링크 및 전화 연결 (url_launcher)
외부 링크를 띄울 때 유용합니다.

활용 사례: "사이트로 이동" 버튼 클릭 시 웹브라우저를 띄웁니다.

패키지의 역할: tel:, https:, mailto: 등 다양한 스키마를 OS가 적절한 앱으로 연결하도록 돕습니다.
