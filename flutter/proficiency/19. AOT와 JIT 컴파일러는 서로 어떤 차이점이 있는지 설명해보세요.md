# AOT와 JIT 컴파일러는 서로 어떤 차이점이 있는지 설명해보세요.

## AOT vs JIT 핵심 비교

| 구분 | JIT (Just-In-Time) | AOT (Ahead-of-Time) |
| --- | --- | --- |
| 컴파일 시점 | 프로그램 실행 중 (Runtime) | 프로그램 실행 전 (Build time) |
| 장점 | 개발 속도가 빠름 (Hot Reload 가능) | 실행 속도가 빠름, 보안에 강함 |
| 단점 | 초기 구동 속도가 느림, CPU 자원 소모 | 빌드 시간이 오래 걸림 |
| Flutter 모드 | Debug 모드 | Release 모드 |
| 결과물 | 바이트코드 (Dart VM에서 실행) | 네이티브 기계어 (ARM, x86 등) |
