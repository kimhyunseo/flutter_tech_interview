# Flutter에서 네이티브 코드와 연동하는 방법에 대해 설명해보세요.
## Flutter에서 네이티브 코드와 연동하는 이유
Flutter는 Dart로 작성되지만
일부 기능은 플랫폼(Android / iOS) 네이티브 API를 직접 사용해야 하는 경우가 있습니다.

### 예
- 블루투스
- 카메라
- 배터리 정보
- 센서
- OS 기능 접근
그래서 Flutter는 네이티브 코드와 통신할 수 있는 구조를 제공합니다.

## 주요 Channel 종류
Flutter에는 몇 가지 채널이 있습니다.

### 채널
- MethodChannel	가장 많이 사용. 함수 호출 방식
- EventChannel	스트림 형태 데이터 전달
- BasicMessageChannel	단순 메시지 교환
실무에서는 대부분 MethodChannel을 사용합니다.

|종류	|특징	|주요 용도|
|-|-|-|
|MethodChannel	|1회성 호출 (Request-Response)	|배터리 잔량 확인, 카메라 열기, 특정 네이티브 함수 실행|
|EventChannel	|지속적인 흐름 (Stream)	|가속도계 센서 값 수신, 실시간 위치 추적, 네트워크 상태 감지|
|BasicMessageChannel	|간단한 메시지나 바이너리 데이터 전송	|문자열이나 대용량 이미지 데이터 전송|
