# JSON이란 무엇인가?
JSON(JavaScript Object Notation)은 데이터를 저장하고 전송하기 위한 경량의 텍스트 기반 데이터 포맷

## JSON의 특징
- 가독성이 높음: 사람이 읽고 쓰기에 매우 직관적
- 가볍다: XML 같은 다른 데이터 형식에 비해 구조가 단순해서 데이터 크기가 작고 전송 속도가 빠름
- 독립적: 특정 언어에 종속되지 않고, 데이터를 주고받는 '형식' 그 자체

## JSON의 기본 구조
JSON은 두 가지 구조를 조합해서 만듬

### 1. 객체 (Object)
중괄호 { }로 묶음
이름(Key) : 값(Value) 의 쌍으로 이루어짐
Key는 반드시 **큰따옴표("")** 문자열로 감싸야 함

### 2. 배열 (Array)
대괄호 [ ]로 묶음
여러 개의 값을 순서대로 나열


## JSON vs XML
| JSON   | XML      |
| ------ | -------- |
| 가볍다    | 무겁다      |
| 파싱 빠름  | 비교적 느림   |
| 태그 없음  | 태그 기반    |
| 가독성 좋음 | 상대적으로 복잡 |


## 직렬화(Serialization)란?
객체를 JSON 같은 문자열 형식으로 변환하는 것

예:
Dart 객체 → JSON 문자열
JSON 문자열 → 객체 (역직렬화, Deserialization)

## 정리
JSON은 데이터를 주고받기 위해 사용하는 경량 데이터 포맷입니다.
키-값 구조로 이루어져 있고, 텍스트 기반이라 가독성이 좋고 파싱 속도도 빠릅니다.
현재 REST API 통신에서 가장 많이 사용되는 형식입니다.
