# Dart 데이터 통신에서의 직렬화-역직렬화에 대해 설명하시오.

## 왜 필요한가?
Dart 객체는 메모리 안에서만 존재

하지만:
서버와 통신하려면
HTTP는 문자열 기반이기 때문에
데이터를 텍스트 형식(JSON) 으로 바꿔야 함

그래서:
Dart 객체 → JSON 문자열 → 서버 전송
서버 응답 JSON → Dart 객체 변환
이 과정이 직렬화 / 역직렬화

직렬화: User 객체 → Map<String, dynamic> → JSON String
역직렬화: JSON String → Map<String, dynamic> → User 객체

## Dart에서 어떻게 구현하나요?
Dart에서는 `dart:convert` 라이브러리를 사용

### 직렬화 (Object → JSON)
```
import 'dart:convert';

Map<String, dynamic> data = {
  'name': 'Hyunseo',
  'age': 29,
};

String jsonString = jsonEncode(data);
```

👉 jsonEncode() 사용

### 역직렬화 (JSON → Object)
```
String response = '{"name":"Hyunseo","age":29}';

Map<String, dynamic> decoded = jsonDecode(response);
```

👉 jsonDecode() 사용

### Model 클래스 사용

#### 모델 클래스 생성
```
class User {
  final String name;
  final int age;

  User({required this.name, required this.age});

  factory User.fromJson(Map<String, dynamic> json) {
    return User(
      name: json['name'],
      age: json['age'],
    );
  }

  Map<String, dynamic> toJson() {
    return {
      'name': name,
      'age': age,
    };
  }
}
```

#### 사용 예
```
// 역직렬화
User user = User.fromJson(jsonDecode(response));

// 직렬화
String jsonString = jsonEncode(user.toJson());
```

## 왜 Model 클래스를 사용하나요?
타입 안정성 확보
가독성 향상
유지보수 쉬움
자동완성 지원

### 자동 생성 방법은?
json_serializable 패키지 사용
freezed + build_runner 사용
실무에서는 수동으로 안 쓰고 코드 생성기 사용함.
